This template is designed for developers who need a stable foundation for creating overlays. The primary focus is on separating system code (renderer) from interface logic.

## Peculiarities
* **Performance**: Minimal overhead, using OpenGL ES 3.2.
* **Easy-to-use**: Adding new ImGui windows takes a couple of lines in `main.cpp`.
